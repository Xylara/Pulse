# Pulse

A encrypted chat application.

> Development is still in progress

## Prerequisites

Before starting, you must ensure you have the following components set up:

1.  Node.js and npm: Installed on your system.
2.  PostgreSQL Database: A running instance of a PostgreSQL database.
3.  Nimbus CDN: This is a CDN made by me for Pulse.
    * The Nimbus repository can be found here: https://github.com/im-playstation451/Nimbus

## Database Setup
> PostgreSQL is required

1.  Configure .env

2.  Run the setup.sql file

## Startup

> You can always use pm2

1.  Install Dependencies:
    `npm install`

2.  Start Pulse:
    `node index.js`
